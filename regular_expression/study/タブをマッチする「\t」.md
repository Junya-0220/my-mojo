
正規表現における「タブ」
正規表現で、タブを表すには、エスケープシーケンスを利用した「\t」を使います。t はタブ（Tab）の頭文字  T なので、覚えやすかと思います。

```perl
# タブを表す正規表現
\t
```

もし、タブ、スペース、改行など、すべての空白文字とよばれる文字をマッチしたいのであれば、「\s」を利用できます。

```perl
# 空白文字を表す正規表現
\s
```

「空白文字」には下記の様な文字が含まれます。

半角スペース（全角スペースを除く）
タブ
改行
改ページ

```perl
my @mocks = (
	"カ ルビ",
	"カ ルビ")

# 正規表現
/^カ\sルビ$/
```

\sならスペース、タブはマッチする。
\tならタブだけマッチする。
